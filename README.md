# XHao_Net_Studio
tools to operate Deep Learning
闲暇之时，捣鼓编程与算法

本产品历时一周不到具体记不得了，主要是想设计一款深度学习之神经网络训练预测的可视化工具，便捷，明了，
初步实现可视化训练及参数设置等

功能：
1、常见的卷积神经网络的训练以及预测（resnet、Unet、alexnet）
2、目标检测的训练与应用（yolo系列）
3、数据集采集、制作

也可以作为qt的练习项目，涉及很多小知识，实属不易呀：
多线程，线程通信，信号与槽函数，matplotlib画图工具嵌入pyqt的简单方法，resnet残差卷积神经网络的模型搭建、训练以及预测的代码，windows屏幕接口（屏幕数据采集）、数据集制作等等
